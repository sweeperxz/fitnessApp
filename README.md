# 🥗 Nutrio — Ваш умный фитнес-ассистент

**Nutrio** — это современное PWA-приложение для отслеживания питания, тренировок и здоровья. Благодаря интеграции с Gemini AI, ваш персональный ассистент всегда готов дать совет по рациону или составить план занятий.

---

## ✨ Основные функции

- 🍎 **Трекер питания**: Контроль КБЖУ (калории, белки, жиры, углеводы), добавление приемов пищи и быстрый поиск по истории.
- 💧 **Учет воды**: Удобный лог потребления воды с постановкой целей.
- 🏋️ **Дневник тренировок**: Планирование и запись упражнений, подходов и весов.
- 🤖 **AI-Ассистент (Gemini)**: Интелектуальный чат, который знает ваши параметры (вес, цели) и дает персонализированные советы.
- 📈 **Аналитика и прогресс**: Визуализация достижений за 30 дней с помощью графиков.
- 📱 **PWA (Progressive Web App)**: Устанавливается как приложение, работает оффлайн и поддерживает пуш-уведомления об обновлениях.
- 🔐 **Безопасность**: Авторизация через Email/Пароль или быстрый вход через Google OAuth.

## 🛠 Технологический стек

### Frontend
- **React 18** + **Vite**
- **Vanilla CSS** (чистый, производительный дизайн)
- **Recharts** (графики прогресса)
- **Vite PWA Plugin** (оффлайн режим и установка)
- **Axios** (API запросы)

### Backend
- **FastAPI** (современный, быстрый Python-фреймворк)
- **PostgreSQL** (надежное хранение данных)
- **SQLAlchemy** (ORM)
- **Alembic** (миграции базы данных)
- **Google OAuth 2.0**

### AI & Services
- **Gemini 1.5 Flash** (AI-модели от Google)
- **Docker & Docker Compose** (контейнеризация)
- **Nginx** (Reverse Proxy)

## 🚀 Быстрый старт (Docker)

Самый простой способ запустить проект — использовать Docker Compose:

1. Клонируйте репозиторий.
2. Создайте файл `.env` в папке `/backend` (используйте `.env.example` как шаблон).
3. Запустите сборку и запуск:
   ```bash
   docker compose up --build
   ```
4. Приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000).

## 🏗 Локальная разработка

### 1. Сервер (Backend)
```bash
cd backend
python -m venv venv
source venv/bin/activate  # или venv\Scripts\activate на Windows
pip install -r requirements.txt
# Настройте .env файл
uvicorn main:app --reload
```

### 2. Клиент (Frontend)
```bash
cd frontend
npm install
# Настройте .env файл
npm run dev
```

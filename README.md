# 🥗 Nutrio — Ваш розумний фітнес-асистент

**Nutrio** — це сучасний і технологічний PWA-додаток (Progressive Web App) для комплексного відстеження вашого способу життя: від детального логування харчування та тренувань до персоналізованих порад від штучного інтелекту.

---

## ✨ Детальний огляд можливостей

### 🍎 Розумне відстеження харчування (Nutrition Tracker)
Nutrio робить контроль раціону максимально простим та інформативним:
- **Контроль КБЖВ**: Миттєвий розрахунок калорій, білків, жирів та вуглеводів для кожного прийому їжі.
- **Гнучкість прийомів**: Додавайте сніданки, обіди, вечері та перекуси.
- **Швидкий пошук**: Зручна база продуктів з можливістю швидкого додавання страв, які ви вживаєте найчастіше.
- **Персональні цілі**: Додаток автоматично підраховує залишок нутрієнтів до вашої денної норми, встановленої у профілі.

### 🏋️ Професійний щоденник тренувань (Workout Diary)
Забудьте про паперові блокноти. Nutrio зберігає всю історію ваших досягнень:
- **Планування занять**: Створюйте тренування на будь-який день тижня.
- **Деталізація вправ**: Записуйте кожну вправу з точною кількістю підходів, повторень та ваги.
- **Історія прогресу**: Переглядайте попередні тренування, щоб відстежити ріст силових показників або зміну інтенсивності.
- **Нотатки**: Додавайте коментарі до тренувань щодо свого самопочуття або особливостей виконання вправ.

### 🤖 Інтелектуальний AI-Асистент (на базі Gemini)
Ваш персональний коуч, який завжди під рукаю:
- **Контекстна обізнаність**: AI знає ваші параметри (вагу, зріст, активність) та цілі (схуднення, підтримка, набір маси).
- **Поради щодо раціону**: Запитайте асистента: "Що мені з'їсти на вечерю, щоб дотиснути норму білка?" або "Склади план харчування на завтра".
- **Відповіді на питання**: Отримуйте науково обґрунтовані відповіді на питання про фітнес, нутриціологію та техніку вправ.

### 💧 Гідратація та здоров'я (Water Tracker)
Підтримка водного балансу — ключ до гарного метаболізму:
- **Логування в один клік**: Швидко додавайте об'єм випитої води.
- **Візуалізація цілі**: Спостерігайте за шкалою прогресу до вашої денної норми (наприклад, 2500 мл).
- **Сповіщення**: Додаток нагадає вам випити склянку води, щоб ви залишалися бадьорими.

### 📈 Глибока аналітика та звіти
Nutrio перетворює ваші дані на зрозумілі графіки:
- **30-денний зріз**: Динаміка споживання калорій та нутрієнтів за останній місяць.
- **Статистика тренувань**: Кількість проведених занять та активність.
- **Аналіз звичок**: Відстежуйте свої "ланцюжки" успіху (streak) — скільки днів поспіль ви дотримуєтеся плану.

### 📱 Переваги PWA та технологій
Nutrio — це більше, ніж сайт. Це додаток майбутнього:
- **Встановлення на головний екран**: Працює без завантаження з App Store чи Google Play.
- **Офлайн режим**: Ваші дані про тренування та воду доступні навіть без інтернету.
- **Пуш-повідомлення**: Отримуйте важливі нагадування про досягнення цілей або нові поради від AI.
- **Google OAuth**: Швидкий та безпечний вхід без необхідності запам'ятовувати паролі.

## 🛠 Технологічний стек

### Frontend
- **React 18** (UI)
- **Vite** (Збірка та HMR)
- **Vanilla CSS** (Легкий та швидкий інтерфейс)
- **Recharts** (Просунута візуалізація даних)
- **Workbox / Vite PWA** (Офлайн можливості)

### Backend
- **FastAPI** (Високопродуктивний Python API)
- **PostgreSQL** (Надійне сховище для ваших даних)
- **SQLAlchemy 2.0** (Сучасний ORM для роботи з БД)
- **Alembic** (Управління міграціями структури БД)

### AI & Infrastructure
- **Gemini 1.5 Flash API** (Могутній інтелект від Google)
- **Docker** (Повна контейнеризація всіх сервісів)
- **Nginx** (Проксі-сервер та статика)

## 🚀 Як почати користуватися

### Варіант 1: Docker (Швидкий запуск)
1. Схиліть репозиторій: `git clone [url]`
2. Налаштуйте `.env` у папці `/backend` (див. шаблон)
3. Запустіть проєкт: `docker compose up --build`

### Варіант 2: Локальна розробка
- **Сервер**: Встановіть Python 3.10+, налаштуйте venv, встановіть `requirements.txt` та запустіть через `uvicorn`.
- **Клієнт**: Встановіть Node.js, виконайте `npm install` та `npm run dev`.

## ⚙️ Змінні оточення (.env)

| Ключ | Призначення |
| :--- | :--- |
| `DATABASE_URL` | Адреса вашої бази даних PostgreSQL |
| `SECRET_KEY` | Рядок для шифрування сесій та токенів |
| `GOOGLE_CLIENT_ID` | Ключ для входу через Google (Google Cloud Console) |
| `GEMINI_API_KEY` | API ключ для роботи штучного інтелекту (Google AI Studio) |
| `VAPID_PUBLIC/PRIVATE_KEY` | Ключі для роботи пуш-повідомлень |
